Bubble Hit Game

I developed an interactive Bubble Hit game using HTML, CSS, and JavaScript. The game challenges players to quickly find and hit a target number displayed on the screen within 30 seconds. The target number changes randomly, and players must search through several bubbles, each containing a different random number, to score points. This project demonstrates my ability to create engaging, time-based interactive games with dynamic content and responsive design.
